SVG

Lagt ut 11.11.2018

SVG (Scalable Vector Graphics) er et XML (eXtensible Markup Language) basert format for å definere vektorbasert grafikk for nettet.

Vektorgrafikk lages ved å plassere punkter på et rutenett og tegne linjer mellom disse punktene. En farge kan så bli lagt til for å fylle ut de linjene som ble laget.

Med vektorgrafikk kan man øke dimensjonen til bildet uten at det påvirker kvaliteten.

SVG er en W3C-standard som integrerer med andre W3C-standarder som DOM og XSL.

Fordeler

SVG er fullt skalerbare, noe som betyr at man kan skalere det opp/ned uten å ofre bildekvaliteten. JPG, PNG og GIF vil begynne å piksellere hvis de strekkes utover den opprinnelige størrelsen.

SVG kan også endre innholdet basert på skjermstørrelse og er følgelig ideelt for bruk i responsivt design.

SVG har god utskriftskvalitet ved enhver oppløsning, og ser skarpt ut på enheter med høy pixel-tetthet.

SVG er tilgjengelig i DOM (Document Object Model) og kan derfor påvirkes med JavaScript, manipuleres med CSS og leses av skjermlesere og søkemotorer

SVG kan manipuleres med CSS eller JavaScript, noe som betyr at man kan nå de ulike delene gjennom klasser, div’er eller tagger, og dermed sette forskjellig stil og oppførsel på dem.

SVG er bare kode og følgelig ideelt format for animasjon. Dette betyr at man kan animere hele bildet som helhet eller bare enkelte deler.

SVG kan opprettes og redigeres med ethvert tekstredigeringsprogram.

SVG kan åpnes med hvilken som helst nettleser.

Bilderedigeringsprogrammer som Photoshop, Sketch, Affinity og Inkscape kan eksportere bilder som SVG slik at ingen kodekunnskap kreves.

Ulemper

SVG virker bare med todimensjonale bilder, som f.eks. logoer, ikoner og illustrasjoner. Fotografier og alt man vil fange med et kamera, virker ikke.

Filstørrelsen kan være stor, selv for små bilder. I tillegg vokser filstørrelsen veldig fort, hvis objektet består av et stort antall små elementer.

Det er litt av en lærekurve for å forstå koden bak SVG og hvordan man skal vise den på nettet på en riktig måte. Dette er i forhold til hvor dypt man vil gå, og om man vil lære å animere det eller manipulere det man får ut. Et komplisert SVG kan også produsere mye kode som det er vanskelig å sile igjennom

Bruksområder

SVG brukes bl.a. for å lage logoer, ikoner, animasjoner, illustrasjoner og skjemaer/grafer for nettsteder.
